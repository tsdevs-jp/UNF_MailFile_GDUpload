# Outlookアドイン「添付ファイル クラウド経由送信」1ページ簡易マニュアル

## 1. 何をするアドインか

Outlook で送る添付ファイルを Google Drive にアップロードし、メール本文に共有リンクを自動で入れて送信するアドインです。

## 2. 使い方

1. Outlook でメールを作成する
2. 送りたいファイルを添付する
3. `送信` を押す
4. 初回だけ Google 認証と保存先フォルダ ID を設定する
5. アップロード完了後、本文に URL が入った状態で自動送信される

## 3. 初回設定

### Google 認証

1. 設定画面で `認証` を押す
2. Google アカウントでサインインする
3. 完了メッセージが出たら閉じる

### 保存先フォルダ ID

Google Drive のフォルダ URL が

```text
https://drive.google.com/drive/folders/1AbCdEfGhIjKlMnOpQrStUvWxYz
```

なら、次の部分を入力します。

```text
1AbCdEfGhIjKlMnOpQrStUvWxYz
```

## 4. 送信時に起こること

- 通常の添付ファイルが Google Drive にアップロードされます
- メール本文の先頭に共有フォルダ URL が追加されます
- 元の添付ファイルはメールから外れます
- 本文内画像などのインライン添付は基本的に残ります

## 5. 受信者に届く内容

相手には、本文の先頭に Google Drive の URL が入ったメールが届きます。添付ファイルそのものではなく、ダウンロード先の案内を送るイメージです。

## 6. うまくいかないとき

### 設定画面が毎回出る

- Google 認証が完了しているか確認する
- フォルダ ID を保存したか確認する

### アップロードに失敗した

- 確認画面で `はい` を押すと、添付付きのまま通常送信できます
- `いいえ` を押すと送信を中止できます

### ログを見たい

設定画面の `ログを開く` から確認できます。

## 7. 注意

- 共有リンクを知っている相手はファイルを参照できます
- 機密ファイルは社内ルールに従って扱ってください

詳しい説明が必要な場合は [取扱説明書.md](C:/Development/UNF_MailFile_GDUpload/取扱説明書.md) を参照してください。
